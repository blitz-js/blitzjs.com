**Alias: `blitz e`**

Exports your Blitz app as a [static application](static-html-export). Make
sure to run `blitz build` before!

#### Options

| Option | Description               | Default |
| ------ | ------------------------- | ------- |
| `-o`   | Set the output directory. | `out/`  |

#### Examples

```bash
blitz build
blitz export
```
