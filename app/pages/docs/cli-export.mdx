**Alias: `blitz e`**

Exports your Blitz app as a static application. Not compatible with
Queries / Mutations! Run `blitz build` before.

[More Information](https://nextjs.org/docs/advanced-features/static-html-export)

#### Options

None

#### Examples

```bash
blitz export
```
