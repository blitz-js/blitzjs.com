## Quick Start

#### Set Up Your Computer

You need Node.js 12 or newer

#### Install Blitz

Run `npm install -g blitz`

#### Create a New App

1. `blitz new myAppName`
2. `cd myAppName`
3. `blitz start`
4. View your baby app at [http://localhost:3000](http://localhost:3000)
