## Auto

Use `h-auto` to let the browser determine the height for the element.

```html amber
<template preview>
  <div class="h-auto w-40 mx-auto rounded-md bg-amber-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-auto</div>
</template>

<div class="**h-auto** ...">h-auto</div>
```

## Screen height

Use `h-screen` to make an element span the entire height of the viewport.

```html rose
<template preview>
  <div class="h-screen mx-auto rounded-md bg-rose-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-screen</div>
</template>

<div class="**h-screen** p-6 ...">h-screen</div>
```

## Fixed height

Use `h-{number}` or `h-px` to set an element to a fixed height.

```html indigo
<template preview>
    <div class="flex justify-around items-end space-between-3">
        <div>
            <div class="h-8 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-8</p>
        </div>
        <div>
            <div class="h-12 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-12</p>
        </div>
        <div>
            <div class="h-16 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-16</p>
        </div>
        <div>
            <div class="h-24 w-8 rounded-md bg-indigo-500 text-white text-2xl font-extrabold"></div>
            <p class="text-center mt-2 text-sm text-indigo-600">h-24</p>
        </div>
    </div>
</template>

<div>
    <div class="**h-8** ..."></div>
    <div class="**h-12** ..."></div>
    <div class="**h-16** ..."></div>
    <div class="**h-24** ..."></div>
</div>
```

## Full height

Use `h-full` to set an element's height to 100% of its parent, as long as the parent has a defined height.

```html
<template preview class="flex justify-around items-end text-sm p-4">
  <div class="h-48">
    <div class="h-full w-40 rounded-md bg-indigo-500 text-white text-2xl font-extrabold flex items-center justify-center p-6">h-full</div>
  </div>
</template>

<div class="h-48">
  <div class="**h-full** ...">h-full</div>
</div>
```

## Responsive

To control the height of an element at a specific breakpoint, add a `{screen}:` prefix to any existing width utility. For example, adding the class `md:h-full` to an element would apply the `h-full` utility at medium screen sizes and above.

```html
<div class="h-8 **md:h-full**"></div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.

## Customizing

### Height Scale

By default, Tailwind's height scale is a combination of the [default spacing scale](/docs/customizing-spacing#default-spacing-scale) as well as some additional values specific to heights.

You can customize the spacing scale for padding, margin, width, and height all at once in the `theme.spacing` section of your `tailwind.config.js` file:

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      spacing: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```

To customize height separately, use the `theme.height` section of your `tailwind.config.js` file.

```diff-js
  // tailwind.config.js
  module.exports = {
    theme: {
      height: {
+       sm: '8px',
+       md: '16px',
+       lg: '24px',
+       xl: '48px',
      }
    }
  }
```
